Inheritance in CSS

In CSS, inheritance controls what happens when no value is specified for a property on an element. When no value for a non-inherited property has been specified on an element, the element gets the initial value of that property. A typical example of an inherited property is the color property. A typical example of a non-inherited property is the border property.

Specificity in CSS

Specificity is the means by which browsers decide which CSS property values are the most relevant to an element and, therefore, will be applied. If the same rule is written twice into the external style sheet, then the lower rule in the style sheet is closer to the element to be styled, and therefore will be applied. A compound selector is a chain of one or more simple (tag, class, ID, universal, attribute) selectors. When you group CSS selectors, you apply the same styles to several different elements without repeating the styles in your stylesheet.

ID and Class Attributes

The only difference between them is that “id” is unique in a page and can only apply to at most one element, while “class” selector can apply to multiple elements. An ID can be used to identify one element, whereas a class can be used to identify more than one.
