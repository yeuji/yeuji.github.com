About Media Queries

Media queries allow webpage content to adapt to different screen sizes and resolutions. Max-width means less than or equal to the width specified in that media query. Min-width means that the style changes even when the screen is larger. We develop for small screens first because the foundation should be focused on the content and mobile design centers on content over navigation. If you focus on the smaller sizes first, then you can create a well-established hierarchy that builds out to be bigger.

Relative Units

Relative units are relative to the size of the parent element's font or the size of the viewport. The relative length units include em, ex, ch, and rem. An em is a unit in the field of typography, equal to the currently specified point size. Percentage units are a main components in responsive web design.

Meta Viewport Tag

We added that meta viewport tag to the top of our pages so that we can specificaly set the initial scale and the device height. The viewport meta tag allows you to use max and min-width that span from desktop to mobile devices. Without the viewport meta tag, my content isn't fitted but with the viewport metea tag, my content is fitted properly on screen.
